职配助手测试反馈清单

请至少测试这些流程：

1. 游客进入 -> 选择求职 -> 粗略诊断
2. 简历经历 -> 使用简历模板或导入 txt/md 简历
3. 填写目标岗位/JD -> 本地分析
4. 查看圆形百分比 -> 点击展开原因
5. 查看“要胜任这个岗位，先补这些”
6. 查看“简历优化执行表”和三份简历草稿
7. 如果有 API Key，设置里选择 DeepSeek/OpenAI 预设，填写 Key 后测试 AI 分析
8. 招聘端 -> 批量粘贴候选人简历 -> 查看排序和面试追问

反馈时请记录：

- 哪一步卡住
- 页面有没有报错
- 你觉得哪个问题不好理解
- 分析结果是否像真实建议
- 简历优化有没有可执行
- AI 输出有没有空泛、跑题或编造数据
